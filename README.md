# `npm` scripts

| Command | Result |
| ---: | :--- |
| `npm run build` | Runs Webpack build with `--mode production` |
| `npm run clean` | Removes `dist` folder |
| `npm start` | Starts `Express` server, alas: `node ./dist/server.js` |
| `npm run start:dev` | Removes dist, builds, and starts Express |
